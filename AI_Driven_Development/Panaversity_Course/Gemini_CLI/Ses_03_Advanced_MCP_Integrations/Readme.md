# Advance MCPs
1) FireBase
  * Firebase is a cloud-based platform that provides a variety of services for building mobile and web applications.
    * Firebase Authentication: Provides a secure and easy way to authenticate users in your app.
    * Firebase Cloud Messaging: Enables real-time communication between your app and Firebase servers.
    * Firebase Realtime Database: Allows you to store and sync data in real-time across multiple devices.
    * Firebase Storage: Provides a simple and secure way to store and serve user-generated content.
    * Firebase Hosting: Enables you to host your static website or web app on Firebase.
    * Firebase Functions: Allows you to write serverless functions that can be triggered by events in your app.
